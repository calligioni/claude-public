# Hooks Magnéticos — Ganchos Estratégicos por Intenção

Skill para gerar hooks poderosos e organizados por intenção estratégica, acabando com o bloqueio criativo de "como começo esse vídeo?".

## O que esta skill faz

Recebe um tema/assunto e entrega hooks divididos em 4 categorias de intenção:
1. **Atração** — hooks que param o scroll e geram curiosidade pura
2. **Autoridade** — hooks que posicionam como especialista/referência
3. **Conexão** — hooks que criam identificação e proximidade com o público
4. **Venda** — hooks que direcionam para ação/conversão (sem ser agressivo)

## Como usar

O usuário informa:
- O **tema** do conteúdo (obrigatório)
- O **nicho** se não for óbvio pelo tema (opcional — perguntar se necessário)
- O **formato** do conteúdo: reels, stories, carrossel, TikTok (opcional — default é reels)
- O **público-alvo** se relevante (opcional)

Se o usuário mandar apenas o tema sem mais detalhes, gerar os hooks direto sem perguntar mais nada. A skill deve ser rápida e prática — o objetivo é desbloquear a criação, não criar mais etapas.

## Regras para Hooks de Qualidade

### Formato
- Todos os hooks em letras minúsculas (padrão Instagram atual)
- Máximo 2 linhas por hook (precisa caber na tela inicial do reels)
- Sem pontuação excessiva (no máximo um "!" quando realmente necessário)
- Sem emojis no hook em si (emojis ficam na legenda, não na tela inicial)
- Linguagem natural, como se estivesse falando com alguém

### O que faz um hook funcionar
Cada hook precisa ativar pelo menos um destes gatilhos mentais:

- **Curiosidade**: a pessoa não consegue seguir sem saber a resposta ("o que ninguém te conta sobre...")
- **Contradição**: desafiar algo que o público acredita ("pare de fazer X se quiser Y")
- **Especificidade**: números e detalhes concretos ("3 coisas que fiz pra triplicar meu faturamento")
- **Identificação**: situação que o público vive ("se você já ficou sem saber o que postar...")
- **Urgência temporal**: sensação de que precisa agir agora ("enquanto você pensa, seu concorrente já...")
- **Resultado tangível**: promessa de transformação concreta ("em 15 minutos você vai ter...")
- **Polêmica leve**: opinião forte mas não ofensiva ("a verdade que ninguém quer ouvir sobre...")

### O que evitar
- Hooks genéricos que servem para qualquer tema ("você precisa saber disso")
- Clickbait vazio que não entrega no conteúdo
- Tom de vendedor agressivo ou infoproduteiro ("compre agora!!!")
- Frases batidas e clichês ("nesse vídeo eu vou te ensinar...")
- Promessas irrealistas ("fique milionário em 7 dias")
- Hooks que precisam de contexto para fazer sentido

### Adaptação por nicho
Os hooks devem ser adaptados ao universo do nicho do usuário. Exemplos de adaptação:

**Marketing digital:**
- Usar termos como engajamento, followers, conversão, conteúdo
- Referenciar plataformas (Instagram, TikTok, YouTube)
- Falar sobre resultados mensuráveis

**Saúde/Bem-estar:**
- Usar linguagem de transformação pessoal
- Referenciar rotina, hábitos, resultados físicos
- Cuidado para não fazer promessas médicas

**Educação/Cursos:**
- Usar termos como aprender, método, processo
- Referenciar dificuldades comuns de aprendizado
- Falar sobre resultados de alunos

**Negócios/Empreendedorismo:**
- Usar termos como faturamento, clientes, escala, processo
- Referenciar desafios reais de quem empreende
- Falar sobre liberdade, tempo, dinheiro

## Funcionalidades extras

### Variações para teste A/B
Se o usuário pedir, gerar 3 variações de um mesmo hook com abordagens diferentes para testar qual performa melhor:

```
🔬 teste a/b para: [hook original]

versão a (curiosidade): [variação]
versão b (autoridade): [variação]
versão c (polêmica): [variação]

💡 sugestão: teste as 3 em stories antes de usar no reels pra ver qual gera mais resposta.
```

### Hook + roteiro rápido
Se o usuário pedir, expandir um hook escolhido em um mini-roteiro de 30 segundos:

```
🎬 mini-roteiro a partir do hook:

📱 tela inicial: [o hook]

🎤 fala (30s):
[script completo do que falar, incluindo abertura, desenvolvimento e CTA]
```

## Checklist de qualidade

Antes de entregar os hooks, verificar:
- [ ] Estão 100% em lowercase?
- [ ] Cada hook tem no máximo 2 linhas?
- [ ] Ativam pelo menos um gatilho mental?
- [ ] São específicos para o tema (não genéricos)?
- [ ] Existe variedade de abordagens dentro de cada categoria?
- [ ] O tom é natural e não soa forçado ou vendedor?
- [ ] Funcionam sozinhos sem contexto adicional?
- [ ] As 4 categorias de intenção estão bem diferenciadas entre si?
